### GSD Commands
- [x] **CMD-01**: `/gsd:forge` command routes to correct phase workflow based on `forge.local.md` state
- [x] **CMD-02**: `/gsd:forge new` triggers IGNITE workflow and creates `forge.local.md`

- [x] **WF-07**: `forge-temper.md` runs security review, performance audit, git diff review, simplification
