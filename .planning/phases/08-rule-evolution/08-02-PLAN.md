```typescript
export interface RegressionDetectorDeps {
  corrections: CorrectionStore;
  repoRoot: string;
  taskNames: string[];  // e.g. ["email_triage", "budget"]
}

export interface RateSnapshot {
  rates: Map<string, number>;  // task_name -> 7-day correction rate
  capturedAt: string;          // ISO 8601
}

export interface RegressionDetail {
  taskName: string;
  rateBefore: number;
  rateAfter: number;
  delta: number;
}

export interface RegressionCheckResult {
  regressed: boolean;
  details: RegressionDetail[];
}
```

Implement `RegressionDetector` class:

**constructor(deps: RegressionDetectorDeps, execFn?: ExecFn)** — stores deps. The `execFn` defaults to a wrapper around `execFileSync('git', args, { cwd: repoRoot })` from `node:child_process`. Use `execFileSync` (NOT `execSync`) to avoid shell injection — following the pattern in dispatcher.ts.

```typescript
// Type for dependency injection in tests
export type ExecFn = (args: string[]) => string;
// Default: (args) => execFileSync('git', args, { cwd: deps.repoRoot }).toString().trim()
```

**snapshotRates(): RateSnapshot** — calls `deps.corrections.rollingCorrectionRate(taskName, 7)` for each task name. Returns a RateSnapshot with the current rates.

**checkForRegression(snapshot: RateSnapshot): RegressionCheckResult** — re-reads current 7-day rates. For each task, computes delta = currentRate - snapshotRate. If any delta > 0, regressed = true. Returns result with all details (including non-regressed for logging).

**revertCommit(commitHash: string): string** — calls execFn with `['revert', '--no-edit', commitHash]`. Returns the new revert commit hash by calling execFn with `['rev-parse', 'HEAD']`. Validate commitHash matches /^[a-f0-9]{7,40}$/ before using. Throws on failure.

**logRegression(result: RegressionCheckResult, commitHash: string, revertHash: string): void** — appends to `{repoRoot}/GROWTH_LOG.md` using `appendFileSync`:
```markdown
## Regression Detected -- {ISO timestamp}

**Reverted commit:** {commitHash} -> {revertHash}
**Affected tasks:**
- {taskName}: {rateBefore.toFixed(3)} -> {rateAfter.toFixed(3)} (+{delta.toFixed(3)})

**Action:** Auto-reverted. Backlog item marked as reverted.
```

**markBacklogReverted(itemDescription: string, reason: string): void** — reads `{repoRoot}/GROWTH_BACKLOG.md` with readFileSync, finds the line containing itemDescription (case-insensitive substring match), appends ` [REVERTED: {reason}]` to that line, writes back with writeFileSync. If file doesn't exist or item not found, log a warning but don't throw.
