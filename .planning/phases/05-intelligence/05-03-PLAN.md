**src/types.ts:**
```typescript
export interface Entity {
  name: string;
  type: string;          // "person", "invoice", "event", "category", "service"
  properties?: Record<string, string | number | boolean>;
}

export interface Relation {
  type: string;          // "SENT_BY", "CATEGORIZED_AS", "ATTENDED", "PAID_FOR", etc.
  properties?: Record<string, string | number | boolean>;
}

export interface Episode {
  subject: Entity;
  relation: Relation;
  object: Entity;
  timestamp?: string;    // ISO 8601, defaults to now
  source?: string;       // "email_triage", "budget_check", "briefing", etc.
}

export interface SearchResult {
  entity: Entity;
  relations: Array<{
    relation: Relation;
    target: Entity;
    timestamp: string;
  }>;
}

export interface KGStats {
  nodeCount: number;
  edgeCount: number;
  staleEdgeCount: number;  // edges older than threshold
}

export interface KGClientConfig {
  uri: string;             // bolt://localhost:7687
  user: string;            // neo4j
  password: string;
  staleThresholdDays?: number;  // default 30
}
```

**src/client.ts:**

Import neo4j-driver. Create `KnowledgeGraphClient` class:

Constructor: Takes `KGClientConfig`. Creates `neo4j.driver(uri, neo4j.auth.basic(user, password))`. Stores staleThresholdDays (default 30).

Methods:

`async addEpisode(episode: Episode): Promise<void>` -- Uses MERGE for both nodes (to avoid duplicates on name+type), CREATE for the relationship edge. Cypher:
```cypher
MERGE (s:Entity {name: $subjectName, type: $subjectType})
SET s += $subjectProps
MERGE (o:Entity {name: $objectName, type: $objectType})
SET o += $objectProps
CREATE (s)-[r:RELATES_TO {type: $relationType, timestamp: $timestamp, source: $source}]->(o)
SET r += $relationProps
```
Use parameterised queries (never string interpolation). Timestamp defaults to `new Date().toISOString()`.

`async search(query: string, type?: string): Promise<SearchResult[]>` -- Case-insensitive partial match on entity name, optional type filter:
```cypher
MATCH (e:Entity)
WHERE toLower(e.name) CONTAINS toLower($query)
  AND ($type IS NULL OR e.type = $type)
OPTIONAL MATCH (e)-[r:RELATES_TO]-(other:Entity)
RETURN e, collect({relation: r, target: other}) as relations
LIMIT 20
```
Map Neo4j records to SearchResult[].

`async getStats(staleThresholdDays?: number): Promise<KGStats>` -- Three count queries:
- `MATCH (n:Entity) RETURN count(n) as nodeCount`
- `MATCH ()-[r:RELATES_TO]-() RETURN count(r) as edgeCount`
- Stale edges: `MATCH ()-[r:RELATES_TO]-() WHERE r.timestamp < $cutoff RETURN count(r) as staleCount` where cutoff = ISO string for (now - thresholdDays)

`async expireStale(olderThanDays?: number): Promise<number>` -- Delete stale edges and orphaned nodes:
```cypher
MATCH ()-[r:RELATES_TO]-()
WHERE r.timestamp < $cutoff
DELETE r
RETURN count(r) as deleted
```
Then clean up orphaned nodes (entities with no remaining relationships):
```cypher
MATCH (n:Entity)
WHERE NOT (n)--()
DELETE n
```
Return the count of deleted edges.

`async close(): Promise<void>` -- `await this.driver.close()`

**Graceful degradation:** Wrap every session operation in try/catch. On neo4j connection errors (ServiceUnavailable, SessionExpired), log a warning and return the graceful default:
- addEpisode: silently fail (log warning)
- search: return empty array
- getStats: return {nodeCount: 0, edgeCount: 0, staleEdgeCount: 0}
- expireStale: return 0
